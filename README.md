# TesseractSample
테서렉트 샘플 만들어봤다

## 목적
안드로이드에서 ocr로 한글을 인식시켜보기 tesseract 라이브러리를 가지고 테스트 프로젝트르 만들어보았다.
구글링해서 어떤 블로그를 참고하여 만들어보았는데 오류도있고 약간의 수정이 필요했다.

## 사용된 기술
1. tesser lib
2. tedbottompicker
3. tedpermission
4. rxandroid
